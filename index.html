Conhecimentos

HTML

HTML (HyperText Markup Language) é a linguagem padrão usada para criar e estruturar páginas web. Ele define a estrutura do conteúdo por meio de tags, como títulos, parágrafos, links, imagens e formulários.

CSS

CSS é uma linguagem usada para estilizar páginas web. Ela define a aparência visual dos elementos HTML, como cores, layouts, fontes e espaçamentos, tornando o design flexível e personalizado.

NodeJs

Node.js é um ambiente de execução JavaScript do lado do servidor, baseado no motor V8 do Chrome. Ele permite criar aplicações escaláveis e de alto desempenho, utilizando programação assíncrona e eventos.

JavaScript

JavaScript é uma linguagem de programação de alto nível criada, a princípio, para ser executada em navegadores e manipular comportamentos de páginas web.

Git e GitHub

Git é um sistema de controle de versão, usado para rastrear mudanças no código. GitHub é uma plataforma de hospedagem para projetos Git, facilitando colaboração e gerenciamento de repositórios online.

React / React Native

O React é uma biblioteca JavaScript usada para construir interfaces de usuário dinâmicas e componentes reutilizáveis. já o React Native é um framework que permite criar aplicativos móveis nativos para iOS e Android usando JavaScript e React.
